This portfolio is intentionally lightweight.

## What I optimize for

1. Fast load time.
2. Easy content updates.
3. Minimal dependencies.

## Workflow

I add a new `.md` file to `src/content/blog` whenever I want to publish a post.
The blog index and post routes are generated automatically during build.
